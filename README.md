# DentaCare Management System

DentaCare Management System is a web-based application for managing dental clinic operations such as patients, dentists, appointments, and billing.

## Technologies Used
- HTML
- CSS
- PHP
- MySQL

## Features
- Add and view patient details
- Add and view dentist details
- Book appointments
- Generate and store billing records
- MySQL database integration

## How to Run
1. Install XAMPP.
2. Copy the project folder into `htdocs`.
3. Start Apache and MySQL.
4. Import `database/dental.sql` in phpMyAdmin.
5. Open `http://localhost/dentacare/` in browser.
